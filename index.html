About This Application

This application was created by Vlad Fedko

This interactive geometric application allows you to explore circle intersections:

  • Click to place points A, B, C, and D on the canvas
  • Two circles will be drawn:
    • Blue circle centered at A with radius AB
    • Yellow circle centered at C with radius CD
  • Drag any point to update the circles in real-time
  • Intersection points (if any) will be shown in red
  • Use the Reset button to start over
